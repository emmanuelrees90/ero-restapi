#ero-restapi

This repository houses a RESTful API built using the latest .NET 7 framework in C#. It's tailored for performance, scalability, and seamless integration with Microsoft SQL Server.

Key Features:

Entity Framework: Utilized for ORM mapping, it provides an efficient way to interact with the database using LINQ, ensuring type safety and reducing runtime errors.
Dapper: An alternative lightweight ORM used in certain modules for performance-critical operations, ensuring rapid data retrieval and manipulation.
SQL Server Integration: Direct integration with MSSQL for robust and scalable data storage.
REST Principles: The API abides by REST guidelines, ensuring stateless communication, standardized routes, and methods.
